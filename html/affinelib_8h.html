Detailed Description

Library for 3D affine transformation.

An implementation of the paper "A concise parametrisation of affine transformation" by S. Kaji and H. Ochiai

LICENSE

the MIT License

Requirements

Eigen library, (optional) MKL

CAUTION

The convention used here is different from the one in the paper; we assume that matrices act on row vectors by right multiplication. ( that is, everything is transposed compared to the paper. )
